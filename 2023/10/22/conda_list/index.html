新的环境：conda create -n *** python=0

5、删除环境：conda remove -n old_name --all # 删除环境old_name

6、激活环境：conda activate xxxx #激活xxxx环境

7、关闭环境：conda deactivate #关闭环境

8、克隆环境：conda create -n new_name --clone old_name

9、升级版本：conda update -n base conda #update最新版本的conda；更新所有的包：conda update --all #更新所有包

==下载东西的时候加镜像：下载使用清华源加上： -i https://pypi.tuna.tsinghua.edu.cn/simple ==

11、pip 管理的常用包

pip search package_name ｜｜pip show package_name ｜｜pip uninstall｜｜package_name｜｜pip list

12、删除环境中的包：conda remove package #删除当前环境中的包

13、在指定环境中安装包：conda install --name env_name package_name

14、删除指定环境中的包：conda remove -- name env_name package

15、使用虚拟环境没有正常推出终端解决方法：# 重新进入虚拟环境 source activate # 退出虚拟环境 conda deactivate

16、conda中一旦进入python之后需要退出的时候输入exit（）
